# RUD-SKILLS

## Purpose
This repository aims to provide users with a platform to add and manage their skills effortlessly using command line utilities.

## How to Use
To add a skill to the RUD-SKILLS repository, use the following command:
```
npx skills add <owner/repo>
```

## Contribution Guidelines
We welcome contributions! If you would like to help improve the RUD-SKILLS project, please follow these guidelines:
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them with clear messages.
4. Submit a pull request detailing your changes.
